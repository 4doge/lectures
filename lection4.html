Компиляция Python

Исходный код написанный на Python компилируется в байт-код для интерпритатора CPython. При этом скомпилированный код мы можем найти в файле с расширением .pyc (этот файл будет создан только если мы вызываем наш скрипт с помощью комманды import)

JIT компиляция - just in time, динамическая компиляция байт-кода в машинный код во время работы скрипта.

Имена

Имя должно начинаться с латинской буквы любого регистра или "_" после чего в имени могут содержаться и цифры(List_of_users, list_of_users, _list_of_user, _list_of_users1). В качестве имени нельзя использовать keywords(которые уже зарезервированы).

Строки документации(docstring)

Python имеет встроенный модуль для документирования pydoc. В начале каждого класса или функции, можно вставить строку документации(docstring). В дальнейшем можно подключить автогенерируемую документацию (swagger, drfdoc etc) или использовать docstring для тестирования - doctest.

GIL, Global Interpreter Lock

Метод синхронизации потоков, который используется в Python.

Создан для того что-бы предотвратить конфликты при одновременном вызове из разных потоков одного и того же участка памяти.

Когда один поток работает, GIL блокирует все остальные.

Конфуз: программа в 1 поток отработает быстрее чем программа в 10 потоков.

Строка

Под строкой подразумевается все что находится между кавычек

Строки которые невозможно изменить всегда помечаются префиксом r


a = "Hello"
b = 'happy'
c = """world"""

escape_string = "\"escaped string here"

qwerty = r"abcd"

print type(a), type(b), type(c), type(qwerty)
full_string = a + b + c
print full_string
print escape_string
# <type 'str'> <type 'str'> <type 'str'> <type 'str'>
# Hellohappyworld
# "escaped string here

Форматирование строк


a = "first"
b = "second"
c = "third"

full_string_concatenate = a + " " + b + " " + c + "!"
full_string_with_format = "{} {} {}{}".format(a, b, c, "!")
full_string_with_num_format = "{0} {1} {2}{3}".format(a, b, c, "!")
full_string_with_key_format = "{key1} {key2} {key3}{key4}".format(key1=a, key2=b, key3=c, key4="!")
full_string_with_all = "{0} {key2} {key3}{key4}".format(a, key2=b, key3=c, key4="!")

print full_string_concatenate
print full_string_with_format
print full_string_with_num_format
print full_string_with_key_format
print full_string_with_all
# first second third!
# first second third!
# first second third!
# first second third!
# first second third!

Словарь

Неотсортированная коллекция элементов, доступ к которой можно получить только по ключу


test_dict = {
    "number": 23,
    2: True,
    "test_list": [1,2,3]
}

print type(test_dict)
print test_dict
print test_dict["number"]
print test_dict[2]

# <type 'dict'>
# {2: True, 'number': 23, 'test_list': [1, 2, 3]}
# 23
# True

Преобразование из одного типа в другой

Для этого достаточно использовать функцию обозначающую тип который Вы хотите получить и передать в нее параметр


num = 1
print type(num), num
num = str(num)
print type(num), num

# <type 'int'> 1
# <type 'str'> 1
